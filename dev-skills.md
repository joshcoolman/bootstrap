# dev-skills

Agent skills worth installing globally. Run any install command to add to all agents on this machine.

## Supabase

| Skill | Installs | When it fires |
|-------|----------|---------------|
| [supabase](https://skills.sh/supabase/agent-skills/supabase) | 143K | Auth, storage, client patterns, migrations |
| [supabase-postgres-best-practices](https://skills.sh/supabase/agent-skills/supabase-postgres-best-practices) | 256K | Schema design, RLS, indexes, functions |

```bash
npx skills add supabase/agent-skills@supabase -g -y
npx skills add supabase/agent-skills@supabase-postgres-best-practices -g -y
```
